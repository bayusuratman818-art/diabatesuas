Project UAS Artificial Intelligence

Deteksi Risiko Diabetes dengan Machine Learning

Sistem pakar berbasis Random Forest Algorithm yang menganalisis rekam medis Anda untuk memberikan prediksi risiko diabetes secara real-time dengan tingkat akurasi tinggi.

Cek Risiko Anda Pelajari Cara Kerja

Teknologi & Fitur Unggulan

Dibangun menggunakan stack teknologi modern untuk performa terbaik.

Random Forest Classifier

Menggunakan model Machine Learning *Ensemble* dari Scikit-Learn yang tahan terhadap *overfitting* dan memberikan akurasi > 90%.

Flask Backend Architecture

Dikembangkan dengan micro-framework Python (Flask) yang ringan, cepat, dan efisien dalam memproses *request* prediksi data.

Probabilistic Output

Tidak hanya memberikan hasil Ya/Tidak, namun juga menyajikan persentase probabilitas risiko berdasarkan fungsi `predict_proba`.

Bagaimana AI Ini Bekerja?

1
Input Data Pasien

Pengguna memasukkan 8 parameter medis seperti Umur, BMI, Level HbA1c, dan Glukosa melalui antarmuka web.

2
Data Preprocessing

Backend Python menggunakan `LabelEncoder` (Joblib) untuk mengubah data kategorikal (Gender, Merokok) menjadi format numerik yang dapat dibaca mesin.

3
Inferensi Machine Learning

Model Random Forest menganalisis pola dari data yang diinputkan berdasarkan dataset pelatihan, lalu menghasilkan prediksi akhir beserta nilai probabilitasnya.
